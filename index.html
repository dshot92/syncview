Keyboard Shortcuts

  • L - Line mode
  • A - Area mode
  • U - Toggle units (metric/imperial)
  • V - Toggle vertex numbers
  • B - Toggle bounding box
  • Delete - Remove last point
  • Ctrl+Z - Undo last point
  • Escape - Close overlays / Clear all

Using Sync View

  • Adding Points: Click on a map to add measurement points
  • Deleting Points: Right-click on any point to delete it
  • Line mode: Measure distances between points
  • Area mode: Measure areas of polygons

About

Sync View is a dual-map comparison tool for measuring distances and areas between locations with real-time synchronization.
